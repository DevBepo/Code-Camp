# Code-Camp

Este repositório reúne os estudos e exercícios realizados no FreeCodeCamp.

## Sobre

Aqui guardo soluções, anotações e projetos práticos do FreeCodeCamp. O objetivo é acompanhar o progresso, revisar conceitos e ter um lugar organizado para compartilhar e consultar o que foi aprendido.

> Observação: como a composição de linguagem do repositório é 100% Python, muitos exemplos e exercícios podem estar em arquivos `.py`.

## Como usar

1. Clone o repositório:

```
git clone https://github.com/DevBepo/Code-Camp.git
```

2. Navegue pelas pastas e abra os arquivos correspondentes.
3. Para rodar exemplos em Python, crie e ative um ambiente virtual (opcional) e execute:

```
python arquivo.py
```

## Contribuições

Pull requests e issues são bem-vindos. Se quiser contribuir, por favor:

1. Abra uma issue descrevendo a mudança proposta.
2. Abra um branch com mudanças pequenas e envie um PR com descrição clara.
